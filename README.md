## Running Project Locally
### Clone the repository
```
git clone https://github.com/LordGhostX/team-x-naijahacks-2019
```

### Install Dependencies
```
pip install -r requirements.txt
```

### Run Project
#### Flask
```
python app.py
```

#### Django
```
python manage.py runserver
```
